# Services Directory

This directory contains infrastructure service interfaces and implementations.

## Contents
- `save_service.dart`: Handles local saving and loading of game states using Hive.
- `audio_service.dart`: Controls music playback, sound effects triggers, fade-in/out, and mute states using FlameAudio.
- `analytics_service.dart`: Tracks gameplay events (level completed, power-ups collected).
- `cloud_sync_service.dart`: Interface for future syncs with remote backends.
