# Engine Directory

This directory contains the Flame game engine integration code, custom components, physics handlers, and camera systems.

## Contents
- `base_game.dart`: Main FlameGame subclass that coordinates the state, inputs, levels, and collision detection.
- `physics/`: Custom gravity, friction, movement momentum, and slope handlers.
- `components/`: Base game components (e.g. player base component, enemy base, breakable tiles, background parallax).
